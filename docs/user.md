# User API Spec

Request Body:

```json
{
  "username": "Krisna",
  "password": "Dwipayana"
  "name": "Krisna Dwipayana"
}
```

Response Body (success):

```json
{
  "data": "OK"
}
```

Response Body (error):

```json
{
  "data": "",
  "errors": "Username Is Blank"
}
```
